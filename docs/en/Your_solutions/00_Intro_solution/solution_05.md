To calculate the horizontal and vertical components of a vector, we use right-triangle trigonometry. The vector $\vec{A}$ acts as the hypotenuse, while its components form the legs of the triangle.

### 1. Vector Decomposition Formulas
Given a vector with magnitude $A$ and an angle $\theta$ measured from the horizontal ($x$-axis):
* **Horizontal Component ($A_x$):** $A \cos(\theta)$
* **Vertical Component ($A_y$):** $A \sin(\theta)$



---

### 2. Step-by-Step Calculation

**Given:**
* Magnitude ($A$) = $15$
* Angle ($\theta$) = $60^\circ$

**Step A: Calculate the Horizontal Component ($A_x$)**
Using the cosine function:
$$A_x = 15 \times \cos(60^\circ)$$
Since $\cos(60^\circ) = 0.5$:
$$A_x = 15 \times 0.5 = \mathbf{7.5}$$

**Step B: Calculate the Vertical Component ($A_y$)**
Using the sine function:
$$A_y = 15 \times \sin(60^\circ)$$
Since $\sin(60^\circ) = \frac{\sqrt{3}}{2} \approx 0.866$:
$$A_y = 15 \times 0.866 \approx \mathbf{12.99}$$

---

### 3. Vector Notation
The complete vector can be written in unit vector notation as:
$$\vec{A} = 7.5\mathbf{\hat{i}} + 12.99\mathbf{\hat{j}}$$

---

### 4. Interpretation and Context
* **Angle Check:** Because the angle is $60^\circ$ (which is greater than $45^\circ$), the vertical component ($12.99$) is significantly larger than the horizontal component ($7.5$). This makes sense physically, as the vector is "leaning" more toward the vertical axis.
* **Pythagorean Theorem Check:** You can verify the result using $A = \sqrt{A_x^2 + A_y^2}$:
    $$\sqrt{7.5^2 + 12.99^2} \approx \sqrt{56.25 + 168.74} = \sqrt{224.99} \approx 15$$
    The math holds up perfectly!

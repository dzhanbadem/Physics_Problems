To find the local maxima or minima of a quadratic function like $f(x) = 3x^2 - 12x + 7$, we use the **First Derivative Test**. This process identifies the "turning point" of the graph.

---

## Step-by-Step Solution

### Step 1: Find the First Derivative $f'(x)$

The derivative tells us the **slope** of the function at any point. We use the power rule:

* The derivative of $3x^2$ is $6x$.
* The derivative of $-12x$ is $-12$.
* The derivative of the constant $7$ is $0$.

$$f'(x) = 6x - 12$$

### Step 2: Find the Critical Point

A local maximum or minimum occurs where the slope of the tangent line is zero (the graph "flattens out"). We set the derivative to zero and solve for $x$:


$$6x - 12 = 0$$

$$6x = 12$$

$$x = 2$$

### Step 3: Determine if it is a Maxima or Minima

To find out if $x=2$ is a peak or a valley, we look at the **Second Derivative** $f''(x)$:


$$f''(x) = \frac{d}{dx}(6x - 12) = 6$$

Since $f''(x) = 6$ (which is a **positive** value), the graph is **concave up** (shaped like a "U"). This confirms that our critical point is a **local minimum**.

### Step 4: Calculate the $y$-coordinate

To find the exact point on the graph, plug $x=2$ back into the **original** function:


$$f(2) = 3(2)^2 - 12(2) + 7$$

$$f(2) = 3(4) - 24 + 7$$

$$f(2) = 12 - 24 + 7$$

$$f(2) = -5$$

---

## Summary of Functions

| Function Name | Equation | Purpose in this Problem |
| --- | --- | --- |
| **Original Function** | $f(x) = 3x^2 - 12x + 7$ | Defines the shape and position of the parabola. |
| **First Derivative** | $f'(x) = 6x - 12$ | Finds the location ($x$) where the slope is zero. |
| **Second Derivative** | $f''(x) = 6$ | Confirms the "curvature" (concavity) of the graph. |

**Final Answer:** There is a **local minimum at (2, -5)**.
